# PAWPACT 毛球盟約

一款原創、零依賴、可離線運行的貓狗混編自走棋瀏覽器 MVP。

## 啟動

```bash
npm run serve
```

打開 `http://127.0.0.1:4173`。

## 驗證

```bash
npm run check
```

這會執行 JavaScript 語法檢查、Node 內建測試及靜態安全／結構檢查。

## 核心玩法

1. 從五張商店卡招募貓狗。
2. 選取備戰區角色，再點我方半場部署；最多上場四隻。
3. 三隻同名同星角色會自動合成升星。
4. 依貓、狗、獵手、護衛與跨物種組合觸發羈絆。
5. 開始對戰後角色會自動移動、攻擊與施放技能。

## 邊界

- 本版本沒有後端、帳號、付費、外部 API 或網路對戰。
- 遊戲狀態只存在目前瀏覽器分頁；重新整理會重開一局。
- 八隻角色都有原創 SVG fallback；橘拳另有 Image Gen 製作的高解析戰鬥動畫包，對戰時會播放普通攻擊、肉球飛撲、受擊與勝利動作。
- 系統偏好「減少動態效果」時會保留 SVG 靜態圖，避免不必要的動作刺激。
